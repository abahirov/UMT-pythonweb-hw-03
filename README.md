# UMT-pythonweb-hw-03

Найпростіший веб-додаток на чистому `http.server` (без фреймворків): дві сторінки, обробка форми, збереження повідомлень у JSON та їх вивід через Jinja2-шаблон.

## Встановлення та запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 main.py
```

Додаток буде доступний на `http://localhost:3000`.

## Маршрути

| Маршрут | Метод | Опис |
|---|---|---|
| `/` | GET | Головна сторінка (`index.html`) |
| `/message.html` | GET | Форма відправки повідомлення |
| `/message` | POST | Обробка даних форми, запис у `storage/data.json` |
| `/read` | GET | Jinja2-шаблон з усіма збереженими повідомленнями |
| `/style.css`, `/logo.png` | GET | Статичні ресурси |

Будь-який інший маршрут повертає сторінку `error.html` зі статусом 404.

## Зберігання повідомлень

Кожне повідомлення з форми (`username`, `message`) дописується у `storage/data.json` під ключем — часом отримання (`datetime.now()`):

```json
{
  "2026-06-30 10:07:08.377072": {
    "username": "krabaton",
    "message": "First message"
  }
}
```
